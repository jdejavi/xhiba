Manual de ayuda de Xhiba

Este es el manual de ayuda al usuario con respecto a los servicios proporcionados por XHIBA.COM

Servicio de correo con RoundCube

Roundcube​ es un cliente de correo para ver los mensajes de correo (correo electrónico) a través de una página web.

Nosotros le ofrecemos un correo personal de xhiba ("example"@xhiba.com) con el que podrá comunicarse sin problemas con otros correos electrónicos (no es necesario que sean de xhiba también).

Blog con Wordpress

WordPress es un sistema de gestión de contenidos (CMS) que permite crear y mantener un blog u otro tipo de web.

Desde xhiba le ofrecemos la posibilidad si quisiera de poseer un blog en Wordpress donde puede subir el contenido que desee. Para acceder a éste, seleccionaremos el emoticono central una vez hayamos iniciado sesión.

Alojamiento web

Ofrecemos a nuestros usuarios la posibilidad de alojar su propia página web en el servidor, siempre que no sobrepase los 5Mb. La descarga y subida de ficheros se realiza por un protocolo seguro, el SFTP.

Para iniciar una conexión al servidor por medio de sftp para subir archivos, se tendría que introducir el siguiente comando en el terminal: sftp "usuario"@xhiba.com Para movernos entre los directorios y ficheros se utilizarán las órdenes de Debian.

Ejemplo subir/bajar archivo:

put [/ruta/a/la/carpeta] "archivo.txt" -> Subir archivo

get [/ruta/a/la/carpeta] "archivo.txt" -> Bajar archivo
